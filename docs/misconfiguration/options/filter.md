```bash
vul conf --severity HIGH,CRITICAL examples/misconf/mixed
```

## By Misconfiguration IDs

Use `.vulignore`.

```bash
$ cat .vulignore
# Accept the risk
AWS003
AWS018
AWS025

$ vul conf --severity HIGH,CRITICAL examples/misconf/mixed
```

## Show Successes and Exceptions
Use `--include-non-failures` option to show successes and exceptions as well as failures.

```bash
vul conf --severity CRITICAL --include-non-failures examples/misconf/mixed
```
